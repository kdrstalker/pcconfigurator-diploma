# Інформаційна технологія підбору конфігурації ПК

Вебзастосунок для автоматизованого підбору комплектуючих з перевіркою сумісності.
Розроблено в рамках магістерської кваліфікаційної роботи.

## Функціонал
- **Ручний конфігуратор:** Покроковий вибір з автоматичною фільтрацією сумісних деталей (Socket, RAM Type).
- **Розумний автопідбір:** Генерація збірки під заданий бюджет та задачу (Ігри, Робота, Рендер).
- **Особистий кабінет:** Збереження та керування збірками.
- **Адмін-панель:** Керування товарами та категоріями.
- **Експорт:** Генерація PDF-звітів.

## Технології
- **Backend:** PHP 8.1 (Native, PDO)
- **Database:** MySQL 8.0
- **Frontend:** HTML5, Bootstrap 5.3, JavaScript
- **Server:** Apache

## Встановлення
1. Імпортувати файл `database/init.sql` в MySQL.
2. Налаштувати підключення в `config/db.php`.
3. Запустити на локальному сервері (Apache/Nginx).
